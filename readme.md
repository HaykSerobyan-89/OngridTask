## *Installation*
### ***Run With Docker Container***
`docker-compose up --build -d`
### *Run from terminal*
#### *To run Django Backend App*
```commandline
pip3 install -r requirements.txt
cd app && python3 manage.py runserver
```

Open new terminal to run React Frontend App
```commandline
yarn install
yarn start
```
